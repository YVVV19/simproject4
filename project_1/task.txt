Мета: Створення функціонального веб-сайту для демонстрації та продажу електронної техніки з можливістю перегляду товарів, оформлення замовлень та комунікації з менеджером.

Цільова аудиторія: Покупці, які шукають техніку онлайн; адміністратори магазину для керування товарами та замовленнями.

Технології: Python, Django (як веб-фреймворк), HTML, CSS, JavaScript (для інтерактиву). База даних SQLite (для простоти, можна розширити до PostgreSQL).
Функціональні вимоги


Управління продукцією (Для адміністратора)
Адмін-панель Django: Повинна використовуватися для додавання, редагування та видалення категорій товарів та самих товарів.

Моделі: Необхідно створити моделі для:

Категорія (наприклад, "Смартфони", "Ноутбуки", "Телевізори").

Товар (назва, опис, ціна, зображення, кількість на складі, категорія).

Перегляд продукції (Для користувача)
Головна сторінка: Відображає список усіх товарів або найпопулярніших товарів.

Сторінка категорії: Відображає товари, що належать до конкретної категорії.

Сторінка товару: Детальний опис одного товару з його характеристиками, ціною та кнопкою "Додати до кошика".


Кошик та оформлення замовлення
Кошик:

Можливість додавати товари до кошика.

Можливість переглядати вміст кошика (список товарів, їх кількість та загальна сума).

Можливість змінювати кількість товарів у кошику або видаляти їх.

Оформлення замовлення:

Форма для введення контактних даних покупця (ім'я, телефон, email, адреса доставки).

Можливість підтвердити замовлення.


Чат з менеджером
Простий чат: Можливість для користувача надіслати повідомлення менеджеру.

Історія повідомлень: Зберігання історії переписки (для демонстрації, можна реалізувати дуже просто, без складних веб-сокетів).

Адмін-панель: Менеджер (адміністратор) повинен мати можливість бачити та відповідати на ці повідомлення через адмін-панель або окрему сторінку


Етапи розробки (Пропонується для організації роботи)
Налаштування проєкту Django: Створення нового проєкту та додатків (shop, orders, chat).
Розробка моделей: Створення моделей Category, Product, Order, OrderItem, Message (для чату). Запуск міграцій.

Адмін-панель: Реєстрація всіх моделей в admin.py для зручного керування даними.

Створення сторінок перегляду: views.py та templates для головної сторінки, сторінок категорій та детальної сторінки товару.

Реалізація кошика: Логіка додавання, перегляду та зміни товарів у кошику (можна використовувати сесії Django).

Оформлення замовлення: Створення форми та логіки збереження замовлення.

Реалізація чату: Створення моделі для повідомлень, форм для відправки та відображення історії.
